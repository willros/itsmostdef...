# It's most def...

### `itsmostdef...` is a python package that fast and simple classifies the most likely content in a `.fastq|.fasta` file.


## Dependencies
* kraken2 (with DB)
* pyfastx 
* pandas

# USAGE

`[USAGE]:   itsmostdef... <fastq_file> <kraken_db> <n_threads> [kraken2_path (default = kraken2)]`
